# Console Quiz Application

Đây là ứng dụng trắc nghiệm chạy trên Console được viết bằng C#.

## Yêu cầu

Cài đặt .NET SDK:\
https://dotnet.microsoft.com/download

Kiểm tra cài đặt: dotnet --version

## Cách chạy

1.  Tải hoặc clone repository
2.  Mở terminal tại thư mục project
3.  Chạy lệnh: dotnet run

Ứng dụng sẽ chạy trong cửa sổ Console.
